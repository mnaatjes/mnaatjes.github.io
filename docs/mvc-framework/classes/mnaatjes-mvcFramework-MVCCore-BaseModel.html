fill()

BaseModel.php : 234

Fills the model's properties with data from an array.

public fill(array<string|int, mixed> $data) : void

This method iterates through an associative array, and for each key, it attempts to set the corresponding public property on the model instance. It uses PHP's Reflection API to ensure that only properties that actually exist on the model are set, preventing "mass assignment" vulnerabilities. Keys in the input array that do not correspond to a property are safely ignored.

Parameters
$data : array<string|int, mixed>

An associative array of data to assign to the model's properties.

Converts the model's private properties into an associative array.

public toArray() : array<string|int, mixed>

This method uses PHP's Reflection API to access all private properties of the child class. It then uses array_reduce to transform these properties into a single associative array where property names are converted from camelCase to snake_case, making them suitable for use with databases or APIs.

The method is a standard pattern for serializing a model's data.
